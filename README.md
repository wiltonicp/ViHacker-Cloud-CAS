### 统一认证中心
统一认证中心，基于SpringBoot2.x、SpringCloud和SpringCloudAlibaba并采用前后端分离的企业级微服务多租户系统架构。并引入组件化的思想实现高内聚低耦合并且高度可配置化，适合学习和企业中使用，并提供应用管理方便第三方系统接入。模块包括:企业级的认证系统、开发平台、应用监控、统一日志、单点登录、Redis分布式缓存、配置中心、分布式任务调度、接口文档、代码生成等。

#### 基于微服务的统一认证中心，持续开发中
